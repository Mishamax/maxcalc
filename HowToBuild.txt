
How to build MaxCalc from source
================================

The latest version of MaxCalc including sources can be downloaded from
http://code.google.com/p/maxcalc/.

To get the latest development version, use Mercurial:

    hg clone https://maxcalc.googlecode.com/hg/ maxcalc


Prerequisites
=============

 * Qt v4.2 or higher (4.4 or higher is recommended to enable all features).
 * Compiler: gcc 4 or Microsoft Visual Studio 2005 / 2008 / 2010.


Variant 1: Using qmake
======================

Change build options, if neccessary, in `maxcalc_config.pri` file.

Execute in "src" directory:

    qmake
    make

If you're using Visual Studio, replace "make" with "nmake"
If you're using MinGW, replace "make" with "mingw32-make".
You can specify build type by adding `CONFIG += debug`, `CONFIG += release`
or `CONFIG += debug_and_release` parameter to `qmake`.
Compiled binaries will be put in `src\debug` or `src\release` directories.


Variant 2: Using CMake
======================

You can also build MaxCalc using CMake v2.4.4 or higher (2.6 / 2.8 recommended).

You need to use separate directory for build:

    cd src
    mkdir build
    cd build
    cmake ..
    make

If you're using Visual Studio, replace `make` with `nmake`.
If you're using MinGW, replace `make` with `mingw32-make`.
To change build options, use `cmake -i ..` instead of plain cmake (or use `cmake-gui`).
You may need to specify generator by adding `-G "<generator>"` (see `cmake -h` for help).
